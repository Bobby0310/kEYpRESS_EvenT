Keyboard keys

NOTE : PRESS ANY KEYBOARD KEY





These are the types of keys.

(Alphabet Key) Any letter typed.

(Number Key) Any Number Typed is a Number Key

(Special Key) Buttons such as "Ctrl,Caps lock,esc, Tab, Alt,(if you have) Widows sign".

(Other Key) includes "shift,enter,Backspace,etc"

Last But Not Least is... Arrowkey. Which as you can guess is any of the arrow keys.
